# 게임 소개

 할로윈 스타일로 즐길 수 있는 길건너 친구들의 모작 게임입니다. 이전 프로젝트에서 기능 추가와 리팩토링을 진행했으며 계속해서 업데이트할 예정입니다.

### 플레이 방법

- 화면을 터치하여 앞으로 전진
- 화면을 왼쪽 또는 오른쪽으로 스와이프하여 좌우 이동
- 움직이는 오브젝트에 충돌하지 않고 최대한 멀리 전진하세요!
- 사탕을 모아 캐릭터를 해금하세요!
